# DealBreaker — AI-Powered M&A Due Diligence Agent

> Five AI analysts. One acquisition decision. Three minutes.

## What it does
DealBreaker deploys five specialized AI agents in parallel to analyze 
any public company as an acquisition target — producing a structured 
risk brief covering financial health, legal exposure, news sentiment, 
competitive position, and people risk.

## Track
Kaggle 5-Day AI Agents Intensive Capstone — Agents for Business

## Tech stack
- Google Agent Development Kit (ADK)
- Multi-agent parallel workflow
- MCP Server for tool access
- Public data: SEC EDGAR, News APIs, Web Search, Glassdoor

## Architecture
Five specialist agents orchestrated in parallel:
1. Financial Analyst Agent — SEC EDGAR filings
2. News & Sentiment Agent — News APIs + web search
3. Legal & Regulatory Agent — Court records + regulatory filings
4. Competitive Intelligence Agent — Web search + competitor data
5. People & Culture Agent — Glassdoor + LinkedIn signals
